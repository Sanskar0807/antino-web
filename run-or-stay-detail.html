Run or Stay

Agensy is compatible with popular WordPress plugins

CASE REVIEW

The Run or Stay Dating app is BRAND NEW!We launched the Run or Stay app in April 2019. We have not been operating as long as most of the mainstream dating apps, so no, we don’t have millions of users on our app yet. We are asking for you to have patience and allow us time to grow.

CASE TIMELINE

2018 - 2019 September

TECHNOLOGIES USED

Android Language - Java | PHP | IOS - Swift | Stripe payment gateway

CHALLENGES

Whatcha Gonna Do functionality like Social media , Feeds. Chat functionality. In the Beginning , We have faced a lot of confusion about how to show user information for a match. One concept is very challenging like if any user is posting their post in Whatcha Gonna do so we need to disable this post after 48 hours.

SOLUTION

For showing user info we have used swipeable card like tinder. For Searching the match, Earlier we used pincode and after we have implemented location (lat ,lang) based search. We have created complete chat functionality with firebase. We have created a Feed page for Social media functionality with different posts like image post , video post, text image post, text video post.
